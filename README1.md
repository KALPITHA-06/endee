# AI RAG Chatbot using Endee

## Overview
This project demonstrates a simple Retrieval Augmented Generation (RAG) chatbot.

## Features
- Accepts user queries
- Retrieves relevant responses from stored data

## Tech Stack
- Python
- NumPy
- Endee (conceptual usage)

## How it works
- Stores text data
- Matches query with relevant content
- Returns best answer

## Run
python main.py
